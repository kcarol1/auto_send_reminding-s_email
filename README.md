# 描述
在Linux服务器上跑深度学习代码，跑完后自动发送邮件提醒
# 设置账户
在send_emial.py中设置账号和授权码（关于授权码如何获得请自行百度）
# 主要功能
程序结束后发送提示，并且记录了**开始时间**、**结束时间**、**总计时间**以及**监测程序的输入文件（可选项并且需要设置要发送的文件路径）**
